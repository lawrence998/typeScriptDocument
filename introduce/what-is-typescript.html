# 什么是 TypeScript

  • TypeScript 是你必须掌握，必须会用，必须学的语言，记住 没有为什么。

TypeScript 是一种由微软开发的自由和开源的编程语言，它是 JavaScript 的一个超集，扩展了 JavaScript 的语法。
